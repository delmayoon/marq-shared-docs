### Q2. Website Content 중복 과금
- Standard Package (Counterfeit/Gray Market)에서 Website Content 채널을 구매하고, Piracy Module에서도 Website Content를 구매하면 채널 가격이 두 번 과금되는 건지?

### Q3. Counterfeit + Gray Market 동시 구매 시 채널 가격 ✅ 답변 완료
- 두 모듈은 각각 모듈 가격($24,000)을 내는 건 맞는데, Standard Marketplaces 채널을 두 모듈 모두에서 사용하면 채널 가격도 두 번 내야 하는지?
- **답변**: 채널 가격은 두 번 과금 X. Channel = 커버리지(어디를 모니터링할지), Module = 제재 수단(어떤 코드로 enforce할지). 채널 가격은 커버리지에 대해 1번만 과금되고, 모듈을 2개 구매하면 모듈 가격만 2배($24K × 2). 같은 채널에 제재 수단이 2개 붙는 구조.
